THE SHED
A tiny co-working / creative space

GADGET is my take on IoT, UI and web-design: all in one package. Initially designed to be a remotely controlled light, GADGET became a educational/playful/artistic project that has been used to teach and inspire people to create cheap urban farming projects on their own.

The project is based on a Wi-Fi compatible Arduino board, which hosts a web server that can be accessed through the browser.

This page provides an overview of the project, described mainly in text.
